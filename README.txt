 RELEASE NOTES:

 * This package includes FS-UAE 3.0.5 Amiga emulator and WHDLoad 18.6
 * Remember to exit the game properly using the right key, otherwise
   saved games won't be saved to the hard disk. Check the launching
   screen to know what is the key to quit the game.
 * FS-UAE config now includes the suggested settings to enable Save
   States for WHDLoad games. Unfortunately it's not 100% working
 * The screen is no longer streched. By default, the emulator shows
   a 692×540 image
 * By default, scanlines are no longer emulated. If you think you
   will miss the Amiga scanlines, you can edit the "Default.fs-uae"
   file you will find in the "fsuae" directory
 * There is always a blank disk in df0 now. Useful if there are games
   that need to save on the floppy disk
 
------------------------------------------------------------------

 USEFUL KEYS:

 F12  ........ Open/Close Menu, use it to Pause or Save State
 F11  ........ Cycle through zoom modes.
 Print Scrn .. Save a screenshot 
 Alt+F4 ...... Quit the emulator
 Alt+Enter ... Toggle fullscreen mode.

------------------------------------------------------------------

CONTROLS
 
 Joystick 1:
	Cursor Keys + Right Ctrl (fire)
	
 Joystick 2 (only for 2 players games):
    W, A, S, D + Left Ctrl
 
 CD32 controls:
	Cursor Keys – Controls the directional pad.
	C – Red
	X – Blue
	D – Green
	S – Yellow
	Return – Play
	Z – Rewind
	A – Forward
	Right Ctrl or Right Alt also controls the red button.

 Note: if you have a controller (e.g. an Xbox controller) connected
 to your PC, this will be used as Joystick1 instead of the keys
 
 You can change the controls editing the file "Default.fs-uae"
